# V2 live build templates

Production pages are generated by `node scripts/build-live-v2.mjs`, which reads **HTML shells** from:

- `scripts/templates/v2/pages/*-test.html`
- `scripts/templates/v2/blog/*-test.html`

The live site (`index.html`, `/products/`, `/blog/`, `/cart`, etc.) does **not** load anything from `scripts/templates/` in the browser. Runtime assets are under `/shared/`, `/scripts/global/`, and `/assets/`.

## Typical workflow

```bash
node scripts/build-live-v2.mjs          # apply v2 shell to live pages + blog-articles.js
node scripts/fix-live-test-links.mjs      # normalize any stray *-test.html URLs in HTML
```

Edit templates in this folder, then re-run the build. Payment checkout logic is preserved from the existing live `pages/shop/payment.html` (PayPal scripts and form markup).
